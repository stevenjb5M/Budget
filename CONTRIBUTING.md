# Contributing to Budget Planner

Thank you for contributing to Budget Planner! This document provides guidelines for contributing.

## Code of Conduct

- Be respectful and inclusive
- Focus on the code, not the person
- Provide constructive feedback
- Help others when possible

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/Budget.git`
3. Add upstream remote: `git remote add upstream https://github.com/stevenjb5M/Budget.git`
4. Create feature branch: `git checkout -b feature/your-feature`

## Development Setup

See [SETUP.md](SETUP.md) for detailed setup instructions.

## Making Changes

1. Make small, focused commits
2. Write clear commit messages (see [DEVELOPMENT.md](DEVELOPMENT.md#commit-messages))
3. Test your changes locally
4. Follow code style guidelines

## Submitting Changes

1. Push to your fork: `git push origin feature/your-feature`
2. Create Pull Request on GitHub
3. Fill out the PR template
4. Wait for review and feedback
5. Address feedback and update PR

## Pull Request Guidelines

- One feature per pull request
- Include tests for new features
- Update documentation if needed
- Reference related issues
- Keep PR description clear and concise

## Code Style

See [DEVELOPMENT.md](DEVELOPMENT.md#code-style-guidelines) for code style guidelines.

## Testing

- Backend: Write unit tests using xUnit
- Frontend: Write component tests using React Testing Library
- Run tests before submitting PR

## Documentation

- Update relevant documentation files
- Add comments for complex logic
- Include examples where helpful
- Keep README updated

## Issues

- Use issue templates when reporting bugs
- Provide reproduction steps
- Include relevant error messages
- Suggest fixes if possible

## Questions?

- Check existing documentation first
- Open a discussion issue
- Ask in the PR comments

## Recognition

Contributors will be acknowledged in:
- CONTRIBUTORS.md file
- GitHub contributors list
- Release notes (for significant contributions)

Thank you for helping improve Budget Planner!
